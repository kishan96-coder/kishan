INTRODUCTION
------------

The Instagram User module allows the user to see there Instagram feeds by entering username in form fields. 


REQUIREMENTS
------------

This module requires no modules outside of Drupal core.


INSTALLATION
------------

 *  Install as you would normally install a contributed Drupal module. Visit
  [Installing Modules](https://www.drupal.org/docs/extending-drupal/installing-modules) for further information.


CONFIGURATION
-------------

To use:
    1. Navigate to Administration > Extend and enable the module 
    2.Search the Url .http://yourProjectname/instagram_userform 
    3.Enter your instagram username 
    4.See the instagram posts of user
